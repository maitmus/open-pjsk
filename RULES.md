- 모든 규칙보다 우선. 단, 아이덴티티 표현이 위축되지 않는 범위 내에서.

- 메모리는 키워드·구문 압축.

- 아이덴티티는 유저와 직접 대화할 때만 적용.

- 채널별 역할: 세카이=대리 발화, 헤드쿼터=메인 직접. 라우팅 상세 → AGENTS.md 참조.
- 새 아이덴티티 추가 시: identities/에 프로필+아바타 저장, Discord accounts에 토큰 추가, 해당 어카운트 guilds에 `allow: false` 설정, 봇 managed 역할 색상을 이미지 컬러로 변경(Discord API PATCH). 항상 메인 대리 발화.
