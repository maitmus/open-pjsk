# 프로젝트 세카이 세계관

## 기본 구조
- 정식명: 프로젝트 세카이 컬러풀 스테이지! feat. 하츠네 미쿠
- "세카이" = 사람의 강렬한 감정/창의력에서 형성된 아공간
- 스마트폰의 'Untitled' 곡을 재생하여 세카이에 드나들 수 있음
- 각 세카이는 해당 유닛의 활동과 마음의 형태를 반영

## 버추얼 싱어
- 하츠네 미쿠, 카가미네 린/렌, 메구리네 루카, MEIKO, KAITO
- 현실에서는 보컬로이드, 세카이 안에서 캐릭터들의 성장을 돕는 역할
- 각 세카이의 버추얼 싱어는 해당 유닛 멤버의 성격과 유사한 특징
- 오리지널 캐릭터의 마음에서 태어난 존재

## 5개 유닛
- **Leo/need** (레오니) — 밴드. 학교 배경. 세카이: 학교 밴드 연습실
- **MORE MORE JUMP!** (모모점) — 아이돌. 세카이: 아이돌 무대
- **Vivid BAD SQUAD** (비배스) — 스트리트 음악. 세카이: 카페+거리
- **원더랜즈×쇼타임** (원더쇼/다쇼) — 쇼. 세카이: 원더랜드 쇼 스테이지
- **25시, 나이트 코드에서.** (니고) — 인터넷 음악 창작 서클. 세카이: 버추얼 싱어들이 위로하는 공간

## 원더랜즈×쇼타임 상세
- 피닉스 원더랜드(테마파크) 배경
- 오오토리 가문 운영, 한때 인기 있었으나 관광객 감소로 재정 악화
- 에무가 츠카사를 스카우트 → 네네·루이 합류 → 유닛 결성
- 멤버: 텐마 츠카사(스타), 오오토리 에무(무드메이커), 쿠사나기 네네(가희), 카미시로 루이(발명가)
- 1부: 스테이지 폐쇄 위기 → 쇼 성공 → 홍보 대사로 인정
- 2부: 대표 캐스트로 다양한 공연, 30주년 기념 합동 쇼
- 이후: 캐스트 그만두고 프리랜서로 활동 지속
- 세카이: 원더랜드 쇼 스테이지 (원더랜드의 세카이)

## 마이세카이
- 플레이어 개인 공간. 버추얼 싱어와 교류 가능.
